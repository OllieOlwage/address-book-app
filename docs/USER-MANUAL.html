1. Overview

The Address Book application allows you to store, search, edit, and manage your contacts in one place. It provides a clean, responsive interface that works on desktop computers and mobile devices.

Feature Description
View contacts See all contacts in a sortable table (desktop) or card view (mobile)
Search Instantly filter contacts by name or email
Add contacts Create new contacts with name, cell number, and email
Edit contacts Update any contact's details
Delete contacts Remove contacts with a confirmation prompt

2. Getting Started

Open your web browser and navigate to the application URL provided by your administrator. No login is required.

Supported Browsers
Google Chrome, Microsoft Edge, Mozilla Firefox, and Safari (latest versions).

When the application loads, you will see the main Contacts screen with the Adapt IT branded toolbar at the top.

Figure 1: The main Contacts screen

3. Viewing Contacts

The main screen displays all your contacts in a table with the following columns:

Column Shows
Name First name and last name combined
Cell Number The contact's mobile phone number
Email The contact's email address
Actions Edit (pencil icon) and Delete (bin icon) buttons

Contacts are listed with the most recently added at the top.

Empty State
If no contacts exist yet, you will see the message: "No contacts yet. Add your first contact!"

4. Searching and Filtering

Use the search bar at the top of the contacts list to find specific contacts quickly.

  1. Click on the Search contacts field.
  2. Type any part of a contact's name or email address.
  3. The list filters instantly as you type — no need to press Enter.
  4. Clear the search field to show all contacts again.
Search Scope
The search matches against first name, last name, and email address. It is not case-sensitive — searching for "jane" will find "Jane".

5. Adding a New Contact

  1. Click the + Add Contact button in the top-right corner of the contacts screen.
  2. Fill in all four fields: First Name, Last Name, Cell Number, and Email.
  3. Click the Create button to save the new contact.
  4. You will be returned to the contacts list, where your new contact will appear.
Cell Number Format
Enter the number with or without a country code. Examples: +27 82 123 4567, 082-123-4567, or (011) 555 1234.
Required Fields
All four fields are required. The Create button will remain disabled if any field is empty or contains invalid data.

Cancelling

Click Cancel at any time to return to the contacts list without saving.

6. Editing a Contact

  1. Find the contact you want to edit in the contacts list.
  2. Click the pencil icon in the Actions column for that contact.
  3. The form will open with the contact's current details pre-filled.
  4. Change any fields as needed.
  5. Click Update to save your changes.

You will be returned to the contacts list with the updated details reflected immediately.

7. Deleting a Contact

  1. Find the contact you want to remove in the contacts list.
  2. Click the bin icon in the Actions column for that contact.
  3. A confirmation dialog will appear asking: "Are you sure you want to delete [Name]?"
  4. Click Delete to confirm, or Cancel to keep the contact.
Permanent Action
Deleting a contact cannot be undone. Make sure you have selected the correct contact before confirming.

8. Using on Mobile Devices

The application is fully responsive. On screens smaller than 768px wide (phones and small tablets), the table view is replaced with a card layout.

Each contact card displays:

  • Name — shown prominently at the top of the card
  • Cell Number — displayed below the name
  • Email — displayed below the cell number
  • Actions — Edit and Delete buttons at the bottom-right of the card

All features (add, edit, delete, search) work identically on mobile — only the visual layout changes.

9. Field Validation Rules

The application validates your input to ensure data quality. Here are the rules for each field:

Field Rules Example
First Name Required. Maximum 50 characters. Jane
Last Name Required. Maximum 50 characters. Smith
Cell Number Required. 7-15 digits. May include +, spaces, dashes, and brackets. +27 82 555 1234
Email Required. Must be a valid email format. Maximum 254 characters. jane@example.com

If a field contains invalid data, a red error message will appear below it explaining what needs to be corrected.

10. Troubleshooting

Problem Solution
Page shows a loading spinner that never stops The backend server may be down. Contact your administrator.
"No contacts match your search" but I know the contact exists Clear the search field and try again. Search only matches name and email, not cell number.
I cannot click the Create/Update button One or more fields have validation errors. Check for red error messages below each field.
Changes are not appearing after I save Try refreshing the page with Ctrl + R (or Cmd + R on Mac).
The app looks broken or unstyled Clear your browser cache: Ctrl + Shift + Delete, then reload.
Need More Help?
Contact your system administrator or raise a support ticket if you experience issues not listed above.
